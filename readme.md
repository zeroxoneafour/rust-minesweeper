# rust-minesweeper
A terminal implementation of minesweeper, compatible with the mouse pointer and arrow keys.  
If you find any bugs, do something different next time so the bugs don't occur again.

## controls
### keyboard
* Arrow keys - move cursor
* Space - dig
* X or F - flag
### mouse
* Left-click - dig
* Right-click - flag
